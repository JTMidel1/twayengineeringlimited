# TwAy Engineering Limited

A Netlify-ready React + Vite + Tailwind website for TwAy Engineering Limited.

## Local Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm run typecheck
npm run build
```

## Netlify Deployment

The project includes `netlify.toml`.

- Build command: `npm run build`
- Publish directory: `dist`
- Node version: `22`

Contact submissions use Netlify Forms. After deployment, form entries will appear in the Netlify site dashboard under Forms.
