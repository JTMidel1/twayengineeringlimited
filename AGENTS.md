# TwAy Engineering Limited Website

This workspace contains a React + Vite corporate website for TwAy Engineering Limited, a multidisciplinary engineering firm in Port Harcourt, Nigeria.

## Project Map

- `src/client/pages/HomePage.tsx` wires the single-page marketing site.
- `src/client/components/site/SiteNav.tsx` and `SiteFooter.tsx` provide the custom marketing layout.
- `src/client/components/site/sections/` contains the page sections: Hero, About, Services, WhyChooseUs, Projects, CompletedProjects, Process, Testimonials, and Contact.
- `src/client/components/site/ContactForm.tsx` handles public inquiries client-side and stores draft submissions in browser storage for a lightweight static-site workflow.
